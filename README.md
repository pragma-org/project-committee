# Pragma Project Committee

This repository contains the governance structure, meeting minutes, and member information for the Pragma Project Committee.

## Overview

The Pragma Project Committee oversees the strategic direction and governance of Pragma projects. This committee operates under the framework defined in the Pragma Development Records (PDRs), specifically [PDR-0011-Project-Committee](https://github.com/pragma-org/PDRs/tree/main/PDR-0011-Project-Committee).
